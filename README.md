# YelpCamp
## Overview
- YelpCamp is a full-stack web application where users can share and review campgrounds.

## Live Demo
- https://dongin-yelpcamp-d46530ff45b3.herokuapp.com/

## Functionality
- Users can view campground information and reviews without logging in
- If a user is logged in, they can create or edit campground information and reviews
- Users can delete campgrounds and reviews that they created

## Tech Stack
- Frontend: EJS, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose
- External API: Mapbox, Cloudinary
- Deployment: Heroku
